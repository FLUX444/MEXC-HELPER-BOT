# Шапка бота

Картинка-шапка показывается при команде `/start` (главное меню).

**Как добавить шапку:** положи в эту папку файл с именем **`header`** и одним из расширений:
- `header.png`
- `header.jpg` или `header.jpeg`
- `header.webp`

Подойдёт также `шапка.png` или `шапка.jpg`. Достаточно одного файла. После перезапуска бота будет использоваться актуальная картинка.
